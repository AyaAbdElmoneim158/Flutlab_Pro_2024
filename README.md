# E-Commerce App

Welcome to the E-Commerce App repository! This Flutter-based app offers a seamless shopping experience with features like product listings, cart management, checkout, and user authentication, all built with MVVM architecture for better maintainability and scalability.

## Features

- **Authentication:** Secure sign-up, login, and user authentication via Firebase.
- **Product Listings:** Display a list of products with filtering and search capabilities.
- **Cart Management:** Add, remove, and view items in the shopping cart.
- **Checkout Process:** Integrated payment gateway for a smooth checkout experience.
- **Order History:** View past orders and track their status.
- **MVVM Architecture:** The app follows the MVVM design pattern for easier maintenance and scalability.
- **Push Notifications:** Uses Firebase Cloud Messaging (FCM) to notify users about order updates and promotions.
- **User Profile:** Allows users to manage their personal information and addresses.

## Dependencies

The project relies on the following key dependencies, each serving a specific purpose:

- **`carousel_slider`**: A widget to implement a carousel slider for images or items.
- **`cloud_firestore`**: A cloud-hosted NoSQL database for storing and retrieving real-time data.
- **`connectivity_plus`**: To check the network status (whether connected to the internet or not).
- **`cupertino_icons`**: Provides iOS-style icons for the app.
- **`firebase_auth`**: Handles user authentication with Firebase (sign-in, sign-up).
- **`firebase_core`**: Required to initialize Firebase in your app.
- **`flutter_native_splash`**: A package to add a splash screen to your app for Android and iOS.
- **`flutter_rating_bar`**: A package to add a rating bar for products or services.
- **`get`**: A popular state management library that simplifies the code structure.
- **`get_storage`**: A lightweight key-value storage solution for Flutter.
- **`google_sign_in`**: Enables Google sign-in functionality.
- **`http`**: A package to make HTTP requests, such as fetching products from an API.
- **`iconsax`**: A collection of modern, consistent icons for your app.
- **`intl`**: Provides internationalization and localization for handling time, dates, and numbers.
- **`logger`**: A simple logging package for debugging purposes.
- **`lottie`**: Adds beautiful animations from Lottie to enhance the user experience.
- **`readmore`**: A widget to create expandable text for displaying large text.
- **`smooth_page_indicator`**: A smooth page indicator for better navigation in carousels.
- **`url_launcher`**: A package to launch URLs in a mobile browser or external apps.
